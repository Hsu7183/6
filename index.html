第 0 層核心公式

做多：O0 >= A + GapMin and O0 <= A + GapMax and L0 <= A - 1，成交價 A，下一根 Open 出場。

做空鏡像：O0 <= A - GapMin and O0 >= A - GapMax and H0 >= A + 1，成交價 A，下一根 Open 出場。

python mtx_research/run_four_layer_matrix.py --outdir report_outputs/four_layer_matrix --progress-every 200000

python mtx_research/run_four_layer_anchor_ratio.py --outdir report_outputs/four_layer_anchor_ratio_20000 --progress-every 200000

python mtx_research/generate_xs_threshold_scripts.py --report-dir report_outputs/four_layer_anchor_ratio_20000 --outdir xs_scripts/anchor_ratio_threshold_24
